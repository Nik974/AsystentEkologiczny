# Asystent Ekologiczny

**Asystent Ekologiczny** to aplikacja mobilna na system Android, zaprojektowana by wspierać użytkowników w prowadzeniu zrównoważonego stylu życia. Łączy dbałość o planetę z realnymi oszczędnościami czasu i pieniędzy.

---

## Kluczowe funkcje

* **Zarządzanie zakupami domowymi** – Inteligentne listy zakupów, które zapobiegają marnowaniu żywności.
* **Kontrola wydatków** – Monitorowanie budżetu domowego z naciskiem na produkty ekologiczne.
* **System odzyskiwania kaucji** – Moduł pomagający śledzić zwrotne opakowania.

## Tech Stack

*   **Język:** Java
*   **Architektura:** MVVM
*   **Baza danych:** SQLite
*   **UI:** XML
